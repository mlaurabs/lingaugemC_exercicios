# Sintaxe e observações

### Tipos de dados em C:
- Tipos primitivos:
    - char:	caracter
    - int: inteiro
    - float:	real de precisão simples
    - double: real de precisao dupla
    - void:	vazio (sem valor)

### Entrada e saída
- import padrão: #include <stdio.h> ---> input e output
